# Časté problémy

## Nejde se připojit k LogBooku
Pokud se Vám nedaří připojit k LogBooku a zdá se, že je nedostupný, zkontrolujte si, že **máte zapnutou a správně nastavenou [VPN](vpn)**. 

Je to raritní, ale někdy se může stát, že se zrovna LogBook server restartuje. Pokud jste si jistí, že VPN funguje správně a LogBook server je dostupný, dejte mu cca pár minut a zkuste to znovu.
